### コミットメッセージ

| 先頭    | 意味                       |
| ------- | -------------------------- |
| add     | 新規（ファイル）機能追加   |
| update  | 機能修正（バグではない）   |
| clean   | 整理（リファクタリング等） |
| fix     | バグ修正                   |
| hotfix  | クリティカルなバグ修正     |
| change  | 仕様変更                   |
| disable | 無効化（コメントアウト等） |
| remove  | 削除（ファイル）           |
| upgrade | バージョンアップ           |
| revert  | 変更取り消し               |
